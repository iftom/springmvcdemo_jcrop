## SpringMVC下用Jcrop预览裁剪上传头像

最近用了Jcrop实现了一个上传头像的功能，希望能帮助到有需要朋友。



项目无需任何配置，即可直接使用。

作为一个Demo，为了方便大家看到效果，我用properties文件存储了上传头像的地址,实际项目中一般是存在数据库中的。
